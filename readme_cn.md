## What's this?
一个基于C#的无人值守Showroom录制机  
支持下载后端：FFmpeg、minyami、minyami-iori

## How to use
将[`example-config.yml`](/example-config.yml)重命名为`config.yml`，将你想监控的用户添加到`Users`里面  
`Downloader`可以选择你喜欢的下载后端，默认为FFmpeg
